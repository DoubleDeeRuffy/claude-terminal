# Archived Requirements: Claude Terminal v1.0 — UX Fixes

**Defined:** 2026-02-24
**Archived:** 2026-02-27
**Core Value:** The terminal and file explorer must behave as users expect from native desktop tools

## v1 Requirements — All Complete

### Terminal Shortcuts

- [x] **TERM-01**: User can copy selected text with Ctrl+C (passes SIGINT to PTY when no selection) — Phase 2
- [x] **TERM-02**: User can paste clipboard content with Ctrl+V — Phase 2
- [x] **TERM-03**: User can jump by word with Ctrl+Arrow (left/right) — Phase 2
- [x] **TERM-04**: User can paste via right-click in terminal — Phase 2
- [x] **TERM-05**: Terminal tab switching remapped from Ctrl+Arrow to Ctrl+Tab/Ctrl+Shift+Tab — Phase 2

### File Explorer

- [x] **FILE-01**: User can see dotfiles and dotfolders in file explorer tree — Phase 1
- [x] **FILE-02**: User can find dotfiles via file search — Phase 1

### Terminal Management

- [x] **TMGR-01**: User can create new terminal via button above tab control — Phase 3

### Session Persistence

- [x] **SESS-01**: Terminal tabs restored with working directories on restart — Phases 4+6
- [x] **SESS-02**: Last opened project restored on restart — Phases 4+6
- [x] **SESS-03**: Terminal session state saved continuously (crash-resilient) — Phases 4+6
- [x] **SESS-04**: Project deletion cleans up saved session data — Phases 4+6

### Explorer State Persistence

- [x] **EXPL-01**: Expanded folders remembered per-project across switches and restarts — Phase 5+5.1
- [x] **EXPL-02**: File explorer panel visibility remembered per-project — Phase 5
- [x] **EXPL-03**: Explorer state saved continuously with debounce (crash-resilient) — Phase 5+5.1
- [x] **EXPL-04**: Explorer state cleaned up on project deletion — Phase 5

## Deferred to Next Milestone

- **Phase 15 requirements**: Projects panel width persistence (unplanned)

## Out of Scope (Unchanged)

| Feature | Reason |
|---------|--------|
| Chat input hotkeys | Only fixing xterm.js terminal, not chat |
| Terminal split panes | Separate feature, not part of this milestone |
| macOS right-click word selection | Windows-first, defer to v2+ |
